DAFTAR FITUR DAN PERBAIKAN
========================================

Fitur Utama:
- Input identitas (periode, nama, NIP, wilayah)
- Input data eviden harian (tanggal, judul konten, foto, like, komen, share)
- Penyimpanan data otomatis ke localStorage dan IndexedDB (untuk foto)
- Tombol Simpan: menyimpan semua data ke localStorage
- Tombol Backup Data: ekspor seluruh data (termasuk foto) ke file JSON
- Tombol Restore Data: impor data dari file backup JSON
- Tombol Simpan ke Word: ekspor laporan ke file Word (.docx) dengan format tabel dan foto
- Tombol PDF: cetak laporan ke PDF (print dialog)
- Tombol Reset: menghapus seluruh data eviden
- Filter data berdasarkan tanggal
- Notifikasi data belum lengkap
- Notifikasi perubahan data yang belum disimpan
- Penambahan dan penghapusan baris eviden secara dinamis
- Validasi input dan peringatan jika data belum lengkap
- Tampilan responsif untuk perangkat mobile
- Fitur restore dan backup mendukung foto (IndexedDB)

Perbaikan & Pengembangan:
- Perbaikan bug pada proses backup/restore data foto
- Penambahan notifikasi perubahan data
- Penambahan validasi input identitas dan eviden
- Penyempurnaan tampilan tombol dan layout agar lebih user-friendly
- Penambahan info "Create by" yang elegan dan otomatis tersembunyi saat print/export
- Optimalisasi penyimpanan data agar tidak terjadi duplikasi
- Penambahan shortcut tombol aksi (tambah/hapus baris)
- Perbaikan kompatibilitas ekspor Word dan PDF
- Penambahan konfirmasi pada aksi hapus data/baris
- Penambahan placeholder dan auto-fill tanggal pada baris baru

Catatan: Daftar ini dapat diperbarui sesuai update aplikasi berikutnya.
